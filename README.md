# Página Día del Padre - René Guzmán

Proyecto listo para subir a GitHub Pages.

## Estructura

- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`
- `assets/img/recuerdos/` (fotos optimizadas)
- `assets/img/recuerdos/thumbs/` (miniaturas)

## Incluye

- Cartas en pergamino.
- Editor de cartas con contraseña:
  - esposa123
  - rene123
  - jesus123
  - diego123
  - padre123
- Álbum de recuerdos tipo baraja/polaroid.
- Lightbox para ver fotos en grande.
- Juego de tiro al blanco.
- Diseño responsive para celular.

## Nota importante

Las ediciones de cartas se guardan en el navegador (`localStorage`). Si cambias de dispositivo, necesitarás editar también desde ese dispositivo o fijar el texto directamente en `index.html`.
